Plugin add in your posts and pages button "I like" by Mail.ru.

== Description ==

Plugin add in your posts and pages button "I like" by Mail.ru. You can configure all options:<br>
display type, button text, width, show text, show friends faces, show in posts or/and on pages.Russian localization download <a href="http://bogutsky.ru">this</a>.<br>
<br>
Плагин добавляет к вашим постам и/или страницам кнопку "Нравится" от Mail.ru. Вы можете настроить следующие опции:<br>
тип отображения, текст кнопки, ширина, показ дополнительного текста, показ фото друзей, показ в постах и/или на страницах. Русскую локализацию можно скачать <a href="http://bogutsky.ru">здесь</a>.<br>

== Installation ==
1. Upload plugin files to the `/wp-content/plugins/` directory.<br>
2. Activate the plugin through the 'Plugins' menu in WordPress.<br>
<br>
1. Загрузите файлы плагина в директорию `/wp-content/plugins/`.<br>
2. Активируйте плагин в панели администратора.<br>
